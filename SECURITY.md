# Security Policy

## Supported Versions

SpeedKalandra is a personal open source project and currently supports the latest public version available on the `main` branch.

Older versions may not receive security fixes unless the issue is severe and practical to patch.

| Version | Supported |
| ------- | --------- |
| Latest / main | Yes |
| Older releases | Best effort |
| Forks or modified builds | No |

## Reporting a Vulnerability

If you discover a security issue in SpeedKalandra, please do not open a public GitHub issue with sensitive details.

Instead, report it privately by contacting the project maintainer.
